# Правила вклада / Contributing

Спасибо за интерес к нашему проекту! Мы рады получать вклады от сообщества.

## Как внести вклад 🤝

### Сообщение об ошибках (Bug Reports)

1. Убедитесь, что ошибка уже не зарегистрирована в Issues
2. Опишите проблему максимально подробно:
   - Какие действия привели к ошибке?
   - Какой ожидаемый результат?
   - Какой фактический результат?
   - Информацию о вашей системе (ОС, версия PHP, браузер)

3. Включите скриншоты или видео если это поможет

### Предложение улучшений (Feature Requests)

1. Откройте Issue и опишите функцию подробно
2. Объясните зачем нужна эта функция
3. Приведите примеры использования

### Pull Requests

1. **Форк репозитория**
```bash
git clone https://github.com/yourusername/social-network.git
cd social-network
git checkout -b feature/your-feature-name
```

2. **Внесите изменения**
   - Придерживайтесь стиля кода проекта
   - Пишите понятные commit messages
   - Добавляйте комментарии к сложному коду

3. **Тестирование**
   - Протестируйте все функции локально
   - Проверьте совместимость в разных браузерах
   - Убедитесь в отсутствии JavaScript ошибок (F12)

4. **Commit и Push**
```bash
git add .
git commit -m "feat: описание вашего изменения"
git push origin feature/your-feature-name
```

5. **Создайте Pull Request**
   - Дайте краткое описание что изменилось
   - Сошлитесь на связанные Issues
   - Дождитесь review от мейнтейнеров

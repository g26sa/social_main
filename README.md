# Социальная сеть / Social Network

Полнофункциональная социальная сеть с поддержкой постов, чата, игр, музыки и видео.

## Возможности 🎯

- **Профили пользователей** - регистрация, авторизация, редактирование профилей
- **Посты и комментарии** - создание и управление пользовательским контентом
- **Чат** - личная переписка между пользователями
- **Медиа** - загрузка и просмотр видео и музыки
- **Игры** - встроенные мини-игры (Крестики-нолики, Змейка, Breakout)
- **Магазин** - система покупок с историей заказов
- **Аватары** - загрузка и управление аватарами профиля

## Требования 📋

- PHP 7.4+
- MySQL 5.7+
- Apache с поддержкой .htaccess (опционально)
- Современный браузер (Chrome, Firefox, Safari, Edge)

## Установка 🚀

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/social-network.git
cd social-network
```

### 2. Конфигурация базы данных

Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

Отредактируйте `.env` с вашими учётными данными MySQL:
```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=social
```

### 3. Создание базы данных

Выполните SQL скрипт инициализации (если предусмотрен):
```sql
CREATE DATABASE social;
USE social;
-- Импортируйте структуру таблиц
```

### 4. Настройка прав доступа

Убедитесь, что директории имеют права на запись:
```bash
chmod -R 755 uploads/
chmod -R 755 avatars/
chmod -R 755 music/uploads/
chmod -R 755 video/uploads/
```

### 5. Запуск проекта

Разместите проект в корне веб-сервера и откройте в браузере:
```
http://localhost/соцсеть
```

## Использование 💡

### Регистрация и вход
1. Перейдите на страницу регистрации
2. Заполните форму и создайте аккаунт
3. Войдите с вашими учётными данными

### Создание поста
1. Перейдите на главную страницу
2. Нажмите "Создать пост"
3. Напишите текст и нажмите "Опубликовать"

### Игры
- **Крестики-нолики**: игра против человека и бота
- **Змейка**: классическая игра со счётом
- **Breakout**: камнебойка

### Магазин
- Просмотр товаров
- Добавление в корзину
- Оформление заказа
- История покупок

## API Endpoints 🔌

Основные эндпоинты находятся в папке `vendor/`:

- `POST /vendor/create_post.php` - создание поста
- `GET /vendor/get_posts.php` - получение постов
- `POST /vendor/add_comment.php` - добавление комментария
- `GET /vendor/get_comments.php` - получение комментариев
- `POST /vendor/send_message.php` - отправка сообщения
- `GET /vendor/get_messages.php` - получение сообщений

## Безопасность 🔒

- Используйте переменные окружения для чувствительных данных
- Никогда не коммитьте файл `.env` с реальными учётными данными
- Валидируйте все входные данные на сервере
- Используйте подготовленные SQL запросы (prepared statements)
- Регулярно обновляйте зависимости

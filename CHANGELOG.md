## [1.0.0] - 2024-04-08

### Added
- Начальная версия проекта
- Система регистрации и авторизации
- Профили пользователей с аватарами
- Система постов и комментариев
- Чат между пользователями
- Мини-игры (Крестики-нолики, Змейка, Breakout)
- Загрузка и просмотр музыки
- Загрузка и просмотр видео
- Система магазина с корзиной и историей заказов
- Поддержка локализации на русском языке

### Technical
- PHP 7.4+ с MySQLi
- Responsive дизайн с CSS3
- Vanilla JavaScript без зависимостей
- jQuery для DOM манипуляции

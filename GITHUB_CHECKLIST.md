## 🌐 GitHub

### Перед публикацией:

1. **Создать репозиторий на GitHub**
   - Не инициализируйте с README (добавим свой)
   - Выберите MIT License при создании (или используйте существующий)
   - Добавьте description: "Full-featured social network PHP application"

2. **Локальная инициализация Git**
   ```bash
   cd /path/to/project
   git init
   git add .
   git commit -m "Initial commit: Social network application"
   git branch -M main
   git remote add origin https://github.com/yourusername/social-network.git
   git push -u origin main
   ```

3. **Настройка репозитория на GitHub**
   - Добавьте topics: `php`, `social-network`, `mysql`, `javascript`
   - Включите "Issues" для багов и предложений
   - Включите "Discussions" для общего обсуждения
   - Настройте branch protection для main

### После публикации:

- [ ] Добавьте shields.io бэджи в README (опционально)
- [ ] Создайте GitHub Pages документацию (опционально)
